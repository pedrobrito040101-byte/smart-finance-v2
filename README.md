Smart Finance V2 💰

Sistema moderno de gestão financeira desenvolvido para auxiliar no controle de receitas, despesas e organização financeira pessoal através de um dashboard intuitivo e responsivo.

🚀 Tecnologias Utilizadas
React.js
TypeScript
TailwindCSS
Node.js
Vite
API Integration
Responsive Design
📌 Funcionalidades

✔️ Dashboard financeiro moderno
✔️ Controle de receitas e despesas
✔️ Visualização de métricas financeiras
✔️ Interface responsiva e otimizada
✔️ Organização de transações
✔️ Estrutura escalável para novas funcionalidades
✔️ Experiência focada em usabilidade
